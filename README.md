# Ruby_Letters

## Getting Started
```bash
  cd ruby_letters
  ruby ruby_letters.rb "Hello World"
```
With customize color

```bash
  cd ruby_letters
  color=blue,white ruby_letters.rb "Hello World"
```
+ blue : text color
+ white : background color
